# React Shop Portfolio project

[Demo](https://nkozlovskaya.github.io/food-list)

В проекте используется Free Meal API (TheMealDB), библиотека для стилизации элементов MaterializeCSS. Для маршрутизации по страницам использовалась библиотека react-router-dom v6
В данном проекте реализован вывод блюд по категориям, маршрутизация на страницу с рецептом конкретного блюда и поиск по категорям блюд
